# Chat メモディレクトリ

このディレクトリには、ユーザーが質問した内容を技術トピックごとに整理したメモが保存されます。

## メモの目的

- 質問内容と回答を記録
- 技術トピックごとに整理
- 後で参照しやすい形で保存

## メモの構成

各技術トピックごとにMarkdownファイルで管理されます。

### 主要な技術トピック

- **HTMX.md** - HTMXに関する質問と回答
- **FastAPI.md** - FastAPIに関する質問と回答
- **Python.md** - Pythonに関する質問と回答
- **SQLite.md** - SQLiteに関する質問と回答
- **データベース設計.md** - データベース設計に関する質問と回答
- **アーキテクチャ設計.md** - アーキテクチャ設計に関する質問と回答
- **UI-UX設計.md** - UI/UX設計に関する質問と回答
- **開発プロセス.md** - 開発プロセスに関する質問と回答
- **その他.md** - その他の質問と回答

## メモの形式

各メモファイルは以下の形式で記録されます：

```markdown
# {技術トピック名}

## {日付} - {質問内容の要約}

### 質問
{ユーザーの質問内容}

### 回答
{回答内容}

### 重要なポイント
- {ポイント1}
- {ポイント2}

### 参考コード例
{コード例}

### 関連トピック
- {関連するトピックへのリンク}
```

## メモの活用方法

- 同じ質問が来た時に参照
- 学習の振り返りに使用
- プロジェクトのドキュメントとして活用
